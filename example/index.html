n8n AI Chat Widget Demo

This is a React-based AI chat widget component that can be embedded on any website.
Click the bubble in the bottom-right corner to start chatting!

Installation

npm install n8n-ai-chat-widget

Basic Usage

import { FloatingChatWidget } from 'n8n-ai-chat-widget';
import 'n8n-ai-chat-widget/dist/FloatingChatWidget.css';

function App() {
  return (
    <FloatingChatWidget
      apiUrl="https://your-n8n.com/webhook/chat"
      title="AI Assistant"
    />
  );
}
